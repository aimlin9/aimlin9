## 🚀 Featured Projects

### 🌿 [AgroSense](https://github.com/aimlin9/agrosense)
> AI-powered crop disease detection for smallholder farmers in Ghana

- Built with **MobileNetV2 transfer learning** on the PlantVillage dataset
- Achieved **97.3% validation accuracy**
- Designed to work on low-bandwidth, low-cost Android devices
- `Python` `TensorFlow` `MobileNetV2` `Computer Vision`

---

### 💰 [FinTrack Ghana (Finance Dashboard)](https://github.com/aimlin9/finance-dashboard)
> Personal finance dashboard that parses Ghanaian bank statements and generates AI-powered spending insights

- Parses PDF/CSV statements from **GCB, Ecobank, and MTN MoMo**
- Categorizes transactions using **NLP with spaCy**
- Generates AI financial insights via **Google Gemini**
- Full-stack: **Django REST Framework + React**
- `Python` `Django` `React` `spaCy` `NLP` `Google Gemini`

---

### 🔐 [AKS-FedAvg](https://github.com/aimlin9/aks-fedavg)
> Research project: Adaptive Key-Size Engineering for Latency-Efficient Paillier-Secured Federated Learning Aggregation

- Explores **privacy-preserving federated learning** with Paillier encryption
- Optimizes key-size to reduce latency in aggregation rounds
- `Python` `Federated Learning` `Cryptography` `Machine Learning`

---

- `Python` `Federated Learning` `Distributed Systems`
